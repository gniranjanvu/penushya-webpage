# Images Directory

## Profile Image

Please place the profile image here as `profile.jpg` or `profile.png`.

The image will be displayed in the Hero section of the homepage with:
- Glowing aura/border effect
- Floating animation
- Circular frame

### Requirements:
- Format: JPG or PNG
- Recommended size: 800x800px or higher (square)
- File name: `profile.jpg` or `profile.png`
